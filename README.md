# weapp-orderManage
业务订单管理系统

============================

########### 环境依赖
无要求

########### 部署步骤
1. 安装 小程序web开发工具
2. 新建项目
3. 项目目录选择此目录（手机预览请填写AppID）
4. 新建完成
